Dasar Teori

Dalam modul ini, kita akan membahas bentuk CSS, elemen layout HTML, dan elemen form HTML5.

CSS (Cascading Style Sheets) menggambarkan bagaimana elemen-elemen HTML ditampilkan pada layar, kertas, atau media lainnya. CSS mempersingkat pekerjaan dengan cara mengendalikan tampilan (layout) beberapa halaman web sekaligus.

Inline CSS

Penggunaan Inline CSS

Inline CSS adalah cara menambahkan style langsung di dalam tag HTML dengan menggunakan atribut `style`. Biasanya digunakan untuk pengaturan cepat atau ketika style hanya diterapkan pada satu elemen saja. 

                Contoh:
                <p style="color: red;">Ini adalah teks merah</p>

Internal CSS

Penggunaan Internal CSS

Digunakan untuk mendefinisikan gaya untuk halaman HTML tunggal. Internal CSS didefinisikan pada bagian `head` dari halaman HTML, dengan elemen style. 

                Contoh:
                <style> 
                    p {
                        color: blue;
                    }
                </style>

External CSS

Penggunaan External CSS

External CSS memungkinkan kita menuliskan style di file terpisah dengan ekstensi `.css`. File ini kemudian di-link ke beberapa halaman HTML, membuat perubahan layout atau tampilan menjadi lebih efisien. 

                Contoh:
                <link rel="stylesheet" href="styles.css">

CSS Fonts

Penggunaan CSS Fonts

Color property mendefinisikan warna teks yang digunakan. Font-family property mendefinisikan huruf (font) yang digunakan. Font-size property mendefinisikan ukuran teks yang digunakan. 

                Contoh:
                <link href="https://fonts.googleapis.com/css2?family=Poppins:wght@400;700&display=swap" rel="stylesheet">
                body {
                    font-family: 'Poppins', sans-serif;
                }

HTML5 `datalist` Elemen

Penggunaan HTML5 `datalist` Elemen

Elemen `datalist` menspesifikasikan daftar dari pilihan yang telah ditentukan (pre-defined ) untuk sebuah elemen `input`. Pengguna akan melihat daftar drop-down dari pilihan yang telah ditentukan sebagai input datanya. Atribut list dari elemen `input`, harus mengacu pada atribut id attribute dari elemen `datalist`. 

                Contoh:
                <label for="browser">Choose your browser:</label>
                <input list="browsers" id="browser" name="browser">
                <datalist id="browsers">
                    <option value="Chrome">
                    <option value="Firefox">
                    <option value="Safari">
                    <option value="Edge">
                    <option value="Opera">
                </datalist>

HTML5 `keygen` Elemen

Penggunaan HTML5 `keygen` Elemen

Tujuan dari elemen `keygen` adalah untuk menyediakan cara yang aman untuk autentikasi para pengguna. Elemen `keygen` menspesifikasikan daerah pembangkit pasangan kunci (key-pair generator field ) dalam form. Ketika form di kirim (submitted), dua kunci (keys) dibangkitkan, satu sebagai private dan satunya sebagai public. private key disimpan secara lokal, dan public key dikirim ke server. 

                Contoh:
                <form action="/action_page.php">
                    <label for="name">Name:</label>
                    <input type="text" id="name" name="name">
                    <keygen name="key">
                    <input type="submit">
                </form>
